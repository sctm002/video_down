# video_down

快手、抖音，短视频批量下载工具。

## 工具使用说明

- 先进入到目标主页，下拉加载所有视频出来

- 然后点击F12，选择作品列表的ul标签内代码，复制出来

- ![第一步](1.png)
- ![抖音](dy.jpg)

- 放入作品读取工具，读取出所有作品地址

- ![第二步](2.png)

- 将作品地址放入批量下载工具，点击"提取视频"，选择下载目录，然后开始批量下载

- ![第三步](3.png)

- （说明：若下载中提示验证码，需手动选择验证码，然后就可以继续下载。）
